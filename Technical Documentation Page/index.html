General Editing

These shortcuts cover fundamental operations that accelerate text manipulation and code organization, forming the basis of efficient coding in VS Code. Mastering these will significantly reduce mouse usage.

Duplicating lines is a common task, especially when creating similar functions or variable definitions. Use the dedicated command to quickly copy and paste the current line.

Core Editing Commands

  • Move the current line up or down: Alt + Up/Down
  • Duplicate the current line: Shift + Alt + Up/Down
  • Delete the current line: Ctrl + Shift + K
  • Toggle line comment: Ctrl + /
  • Block comment selection: Shift + Alt + A
  • Format the entire document: Shift + Alt + F

For block manipulation, VS Code allows easy indentation adjustments. Use the tab or dedicated command for controlled structure changes.

Example of line duplication and movement const initialValue = 0; const finalValue = 100; Duplicate this line with Shift + Alt + Down

A crucial feature for rapid code changes is renaming symbols across the entire scope, which is typically faster than find-and-replace. This is covered in the Refactoring section.

File and Window Management

Efficient navigation between files and views is critical for large projects. These shortcuts allow you to quickly open, close, and arrange your editor windows.

Switching files without using the mouse is handled by the "Go to File" command, which uses fuzzy search for fast access. This significantly boosts productivity.

Command for opening the Command Palette Ctrl + Shift + P

Split-view editing is essential for reviewing different parts of a file or comparing two distinct files side-by-side. The following command splits the current editor.

Window Control

  • Open a new window/instance: Ctrl + Shift + N
  • Close the current window/instance: Ctrl + Shift + W
  • Split the editor: Ctrl + \
  • Focus next/previous editor group: Ctrl + K, Ctrl + Right/Left
  • Toggle the integrated terminal: Ctrl + `
  • Close current file tab: Ctrl + W
Navigation and Search

Locating specific code or files quickly is a huge time-saver. VS Code provides powerful search capabilities that can span across files, or just within the current document. These shortcuts are essential for project traversal.

The global search feature allows you to find text across every file in your current workspace, supporting regular expressions and case sensitivity. The "Go to File" command is the fastest way to open files.

Global search across files Ctrl + Shift + F

Jumping to definitions or references is crucial for understanding code flow. VS Code uses language server data to enable intelligent navigation.

Search and Symbol Navigation

  • Go to File (Quick Open): Ctrl + P
  • Find in current file: Ctrl + F
  • Go to Definition of symbol: F12
  • Peek Definition (inline preview): Alt + F12
  • Go to Symbol in File: Ctrl + Shift + O
  • View all References: Shift + F12
  • Navigate back/forward: Alt + Left/Right

Use Ctrl + T to search for symbols across all open workspaces and files.

Debugging Controls

The integrated debugger is a powerful feature for analyzing runtime behavior. Shortcuts make controlling the execution flow seamless. Use the Run and Debug view (Ctrl + Shift + D) to manage breakpoints and variables.

Before starting, ensure your launch configuration is set up correctly in the .vscode/launch.json file. Starting the debug session is a single keypress.

Step Over command F10

Setting and removing breakpoints is an essential part of the debugging workflow. This action can be toggled directly on the line of code.

Debug Execution Flow

  • Start/Continue Debugging: F5
  • Toggle Breakpoint: F9
  • Stop Debugging: Shift + F5
  • Step Into function call: F11
  • Step Out of function call: Shift + F11
Multi-Cursor and Selection

Multi-cursor editing is one of VS Code's most beloved features, allowing simultaneous editing at multiple points. This massively speeds up repetitive changes across multiple lines.

The primary method involves adding a cursor above or below the current one, enabling column-like editing across different lines.

Add cursor above or below Alt + Click (or Ctrl + Alt + Up/Down on some systems)

The "Add Cursor to Next Occurrence" commands are the fastest way to target specific instances of a word without selecting every instance globally.

Selection Techniques

  • Add cursor at selection: Alt + Click
  • Select next occurrence of current word: Ctrl + D
  • Select all occurrences of current word: Ctrl + Shift + L
  • Expand/Shrink selection scope: Shift + Alt + Right/Left

Using Ctrl + D repeatedly allows you to select specific, non-contiguous instances of a variable or word for simultaneous editing.

Terminal and Tasks

The integrated terminal simplifies running shell commands and project tasks without leaving the editor. Efficient control over the terminal is vital for build, test, and run cycles.

The main shortcut toggles the terminal's visibility, making it easy to hide or show as needed.

Toggle the integrated terminal Ctrl + `

You can manage multiple terminal instances, navigate between them, and quickly create new ones.

Terminal Control

  • Create New Terminal: Ctrl + Shift + `
  • Focus Terminal: Ctrl + `
  • Run Build Task: Ctrl + Shift + B
  • Kill the active terminal: Ctrl + K, Ctrl + H (or use Command Palette)

Configuring a default build task allows you to launch common tasks like compiling or linting with a single keypress.

Code Refactoring

Refactoring shortcuts help you restructure your code safely and quickly, improving readability and maintainability. These features rely on language server support for accuracy.

The primary refactoring command, "Rename Symbol," is one of the most frequently used shortcuts.

Rename Symbol globally F2

The "Quick Fix" menu provides context-aware refactoring options, such as extracting functions or variables, importing missing dependencies, or fixing syntax errors.

Refactoring and Quick Fixes

  • Rename Symbol: F2
  • Quick Fix/Show potential fixes: Ctrl + .
  • Go to next Error or Warning: F8
  • Go to previous Error or Warning: Shift + F8

Using Ctrl + . is indispensable for modern coding, instantly suggesting solutions for code smells or missing imports.
