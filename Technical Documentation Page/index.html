introduction to java

Java is a high-level, class-based, object-oriented programming language. It is used to develop desktop and mobile applications, web applications, and much more.

One of the key features of Java is its "write once, run anywhere" philosophy. This is achieved through bytecode compilation, where Java source code is compiled into bytecode that can be executed on any JVM.

To get started with Java, you need to install the Java Development Kit (JDK). Here's a simple example of a Java program:

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } }

Java supports various paradigms, including procedural and functional programming. It is widely used for web development, mobile apps (Android), and enterprise applications.

  • Platform-independent
  • Object-oriented
  • Automatic memory management
Classes and Objects

In Java, a class is a blueprint for creating objects. It defines the properties (fields) and behaviors (methods) that objects of that class will have.

An object is an instance of a class. You can create multiple objects from the same class, each with its own state.

class Car { String model; void drive() { System.out.println("Driving"); } }

To create an object, use the 'new' keyword: Car myCar = new Car();

Classes can have constructors, which are special methods called when an object is instantiated.

  1. Define the class
  2. Create object
  3. Access fields and methods
data types and variables

Java has two main categories of data types: primitive and reference. Primitive types include int, double, boolean, and char.

Variables are used to store data. You must declare a variable with its type before using it.

int age = 25; double price = 19.99; boolean isStudent = true;

Reference types include classes, arrays, and interfaces. Strings are a common reference type.

Java is statically typed, meaning variable types are checked at compile time.

  • Primitive types: byte, short, int, long, float, double, char, boolean
  • Reference types: objects, arrays
control flow statements

Control flow statements allow you to control the execution order of your program. Common ones include if-else, switch, for, while, and do-while.

The if statement executes code based on a condition: if (age > 18) { System.out.println("Adult"); }

Loops like for are used for repetition: for (int i = 0; i < 10; i++) { System.out.println(i); }

The switch statement is useful for multiple conditions.

  1. Conditional statements: if, else, switch
  2. Loops: for, while, do-while
Exception Handling

Exceptions are events that occur during program execution that disrupt the normal flow. Java provides try-catch blocks to handle them.

Use try to enclose risky code, and catch to handle exceptions: try { int result = 10 / 0; } catch (ArithmeticException e) { System.out.println("Division by zero"); }

The finally block executes regardless of whether an exception occurs.

Custom exceptions can be created by extending the Exception class.

  • Checked exceptions: must be handled or declared
  • Unchecked exceptions: runtime exceptions
